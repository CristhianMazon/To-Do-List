## 📱 Descrição

Aplicativo de lista de tarefas (To-Do List) desenvolvido em Flutter.
Permite adicionar, marcar como concluídas e remover tarefas, utilizando gerenciamento de estado com `StatefulWidget` e `setState()`.

---

## 🎯 Objetivo

Aplicar conceitos básicos de Flutter:

* StatefulWidget
* setState()
* Manipulação de listas
* Entrada de dados com TextField
* Atualização dinâmica da interface

---

## ⚙️ Funcionalidades

✅ Adicionar tarefas
✅ Exibir lista de tarefas
✅ Marcar tarefas como concluídas
✅ Remover tarefas
✅ Limpar campo após adicionar
✅ Mensagem quando a lista está vazia

## ▶️ Como Executar

1. Clone o repositório:

```
git clone https://github.com/seu-usuario/todo-list-flutter-seunome.git
```

2. Acesse a pasta:

```
cd todo-list-flutter-seunome
```

3. Instale as dependências:

```
flutter pub get
```

4. Execute o projeto:

```
flutter run
```

---

## 💡 Observações

* O estado da aplicação é gerenciado com `setState()`
* As tarefas são armazenadas em uma lista (`List<Tarefa>`)
* O campo de texto é controlado por `TextEditingController`
* O método `dispose()` é utilizado para liberar recursos corretamente
